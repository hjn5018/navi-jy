# 🚀 NAVI 향후 고도화 로드맵 (Future Roadmap)

현재 NAVI는 핵심 파이프라인(의도 분석 -> 에이전트 실행 -> UI 출력)이 완료된 상태입니다. 다음은 더 실용적이고 강력한 AI 비서가 되기 위한 후속 작업 리스트입니다.

## 1. 에이전트 기능 고도화
- **브라우저 액션 확장**: 단순히 검색 결과를 긁어오는 것을 넘어, 특정 영상 클릭, 댓글 읽기, 로그인 기반 서비스(메일, 캘린더) 제어.
- **다중 OS 제어**: 파일 탐색기 연동(파일 복사/이동), 시스템 설정 변경(볼륨 조절, 다크 모드 전환), 오피스 프로그램(Excel, PPT) 자동화.
- **멀티 에이전트 협업**: "메모장에 적힌 내용을 요약해서 메일로 보내줘"와 같은 복합 작업 수행 능력 강화.

## 2. 사용자 경험(UX) 및 인터페이스
- **멀티모달 입력**: 음성뿐만 아니라 화면 캡처 이미지를 보고 질문에 답하는 기능(Vision API 연동).
- **개인화 학습**: 사용자의 선호 채널, 자주 사용하는 프로그램 등을 학습하여 추천 정확도 향상.
- **실시간 터미널 위젯**: 에이전트가 수행 중인 실제 명령(CLI)이나 브라우저 로그를 실시간으로 볼 수 있는 디버그 뷰 추가.

## 3. 기술적 최적화
- **로컬 LLM 지원**: 개인정보 보호를 위해 Ollama 등을 이용한 로컬 언어 모델(Llama 3 등) 연동 옵션 제공.
- **지연 시간(Latency) 극소화**: Streaming STT 연동(사용자가 말하는 도중에 의도 파악 시작)을 통한 응답 속도 체감 단축.
- **에러 복구 로직**: Playwright 작업 중 버튼을 못 찾을 경우 스스로 대안을 찾는 Self-healing 에이전트 구현.

## 4. 단일 설치 파일 (Standalone EXE) 및 배포용 고도화
- **Zero-Dependency 배포 모델**: 사용자가 Python, Node.js 등을 직접 설치하지 않아도 되도록 모든 엔진을 하나의 `.exe` 또는 설치 관리자(Installer)에 포함.
- **PyInstaller 통합**: 백엔드(FastAPI)와 런타임을 하나의 실행 파일로 묶어 별도 환경 구성 없이 즉시 실행 가능하도록 패킹.
- **환경 자가 진단 (Env Health Check)**: 프로그램 첫 실행 시 마이크 권한, 네트워크 상태, API Key 유효성을 자동으로 체크하고 가이드 제공.
- **자동 업데이트**: 배포 후 새로운 에이전트 기능이나 보안 패치를 실시간으로 적용할 수 있는 자동 업데이트(OTA) 시스템 구축.

## 5. 확장성 및 통합
- **모바일 원격 제어**: 모바일 앱에서 PC의 NAVI를 제어하는 기능.
- **Smart Home 통합**: Home Assistant 등과의 연동을 통한 IoT 기기 음성 제어.
