## 프로젝트 명
NAVI

## 프로젝트 설명
시각 장애인, 저시력자, 일반 사용자를 위한 음성 명령 기반 브라우저 및 windows 데스크톱 제어 프로그램

## 특징
백그라운드 상시 감지 프로그램이 단축키나 웨이크 워드로 메인 프로그램을 실행한다.
playwright, windows ui automation으로 chrome browser 실행, 유튜브 검색, 네이버 경로 탐색, 메모장 입력 및 저장 동작 등을 실현한다.

## 주요 기능
- windows 시작 시 백그라운드 프로그램 자동 실행, 호출 감지 시 메인 프로그램 실행
- 자연어 음성 명령 처리를 위해 사용자 음성을 STT변환
- LLM을 통해 의도 분석 및 작업 계획 생성, Agent 할당
- 웹 브라우저 서비스 또는 windows 프로그램을 실행 및 조작한다.
- 실행 결과를 검증하고, 예외 처리를 한다.
- 수행 결과를 요약하여 자연어로 안내하고, 후속 행동을 질의한다.
- windows ui automation으로 현재 ui 상태를 파악하고, 비밀번호와 같은 개인 정보는 음성 처리 및 로그에 제한을 둔다.
- 후속 질문("첫 번째 영상을 재생할까요?")으로 다음 동작을 유도한다.

## 예상 적용 기술
- GPT-4.1 mini
	- 사용자의 음성 텍스트 명령을 이해하고 작업 계획을 생성
- Playwright 
	- 웹 브라우저를 제어 
- Windows UI Automation 
	-  Windows 프로그램 제어
- gpt-4o-mini-transcribe
	- STT
- GCP Text-to-Speech
	- TTS

## 주요 시나리오
- 유튜브
	- 사용자 명령
		- "유튜브에서 고양이가 춤추는 영상 찾아줘"
	- 시스템 동작
		- 브라우저 실행
		- 유튜브 접속
		- 검색어 입력
		- 결과 수집
		- "첫 번째 영상은 ..이며, 재생시간은 ..분입니다." 안내
- 네이버 지도 길 찾기
	- 사용자 명령
		- "네이버 지도로 서울역에서 한국폴리텍대학 인천캠퍼스로 가는 지하철 경로 찾아줘"
	- 시스템 동작
		- 김찾기 경로 탐색
		- 지하철 모드 선택
		- 소요 시간 및 환승 정보 추출
		- 후속 질문 생성
