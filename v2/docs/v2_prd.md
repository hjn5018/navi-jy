# NAVI V2 제품 요구 사항 (PRD)

## 1. 개요 (Overview)
NAVI V2는 사용자의 음성 및 텍스트 명령을 이해하고 웹(YouTube 등) 탐색 및 윈도우 OS 프로그램(메모장 등) 제어를 수행하는 로컬 기반 AI 어시스턴트입니다. V1에 존재했던 API 비용 문제와 음성 인식 오류, 그리고 부정확한 명령어 파싱으로 인한 동작 실패 문제를 해결하기 위해 **완전한 로컬 환경(Local AI)**과 **함수 호출(Function Calling) 기반의 에이전트 아키텍처**를 도입합니다.

## 2. 해결 목표 (Problem to Solve)
- **API 종속성 및 비용 제거**: OpenAI 등의 외부 API에 의존하지 않고 로컬 환경에서 STT, TTS, LLM을 구동합니다.
- **음성 인식(STT) 성능 개선**: 사용자의 한국어/영어 명령을 더 정확하고 빠르게 인식해야 합니다.
- **명령 이해 및 실행 정확성 향상(오류 보정)**: 음성 인식이 일부 잘못되더라도 문맥을 파악하여 올바른 액션(브라우저/OS 제어)을 실행해야 합니다.

## 3. 핵심 기능 (Key Features)

### 3.1 로컬 음성 처리 (Local STT & TTS)
- **고성능 로컬 STT**: 마이크로 입력된 사용자 음성을 빠르고 정확하게 텍스트로 변환.
- **로컬 TTS 응답**: 에이전트의 답변을 자연스러운 음성으로 출력.

### 3.2 로컬 LLM 및 인텐트 파싱 (Function Calling)
- **문맥 기반 오타 교정 및 의도 파악**: STT 결과에 오타가 있더라도 LLM이 문맥을 이해하여 교정합니다. (예: "냅모장 열어줘" -> "메모장 실행")
- **도구 호출(Tool Calling/Function Calling)**: 자연어를 분석하여 미리 정의된 시스템 제어(OS, Web) 함수를 호출하는 인자를 정확하게 생성.

### 3.3 웹 및 OS 자동화 에이전트 (Web & OS Automation)
- **웹 자동화 (Playwright 기반)**: YouTube 검색, 웹사이트 네비게이션, 정보 추출.
- **OS 제어 (uiautomation/pyautogui 활용)**: 메모장, 계산기, 특정 프로그램 실행 및 시스템 제어.
- **다중 동작 지원**: "메모장 열고 유튜브에서 음악 틀어줘" 등 복합 명령 처리.

## 4. 비기능적 요구 사항 (Non-Functional Requirements)
- **퍼포먼스(지연 시간)**: 음성 명령 종료 후 첫 응답 및 액션 실행까지 최대 2-3초 이내.
- **오프라인 구동**: 앱 실행 시 인터넷 연결 없이도(웹 검색 기능 제외) 시스템 제어 및 일상 대화 가능.
- **확장성**: 추후 새로운 도구(기능)를 쉽게 추가할 수 있는 플러그인 구조.

## 5. 제외 범위 (Out of Scope for initial V2 Release)
- 모바일(iOS/Android) 전용 앱 (PC 데스크톱 환경 우선).
- Mac 환경 완벽 지원 (Windows OS 제어를 우선).
