# 배경
- 사용자의 음성 및 텍스트 명령을 실시간으로 분석하여 웹 탐색(YouTube 등)과 Windows OS 제어(메모장 입력 등)를 자동화하는 프로그램을 NAVI를 만들었어.  
# 문제
- 기본적으로 api 비용이 들지 않도록 로컬로 구동이 가능하게 만들었어.  
- tts 응답이 나오기는 하는데, 음성 인식이 좋지 않아.
- 일부 음성 인식이 되더라도 windows 프로그램 실행이나 브라우저 실행과 같은 동작이 발생하지 않는 것 같아.
# v2 개선
- 이전 NAVI 프로그램을 v1으로 하고, v2를 만들고 싶어.
- 사용 가능한 프레임워크, 기술 스택, 라이브러리 등을 여러 조합으로 고안해보고, 가장 나은 조합을 선택해줘.
- v1의 소스 코드와 문서들은 참고만 하고 새롭게 문서들부터 만들고 시작해줘.
